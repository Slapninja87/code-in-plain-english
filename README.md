# code-in-plain-english
These focus on stripping away the jargon in coding and programming and making people feel welcome

**Web & Frontend Development**: These languages power the visual elements, styles, and interactive behaviors of modern websites.

  JavaScript: The undisputed language of the web, used to build interactive websites.

  TypeScript: A strongly typed superset of JavaScript built by Microsoft for large-scale web apps.

  HTML: The standard markup language used to structure web pages.

  CSS: The style sheet language used to format the layout and design of web pages.


**Backend & Server-Side Development**: These languages run on servers to process data, manage user accounts, and power business logic behind the scenes.

  Python: Highly popular for its simple, English-like syntax and vast ecosystem.

  Java: A versatile, object-oriented language known for its "write once, run anywhere" portability.

  C#: Developed by Microsoft, heavily used for enterprise software and web backends.
  
  PHP: A server-side scripting language that powers a massive portion of web content, including WordPress.

  Ruby: Famous for its elegant readability and the robust Ruby on Rails web framework


**Mobile App Development**: Engineers use these specific languages to build application software for smartphones and tablets.

  Swift: Apple's modern, fast, and secure language optimized for iOS and macOS apps.

  Kotlin: Google's preferred, modern programming language for native Android application development.

  Dart: Developed by Google to power Flutter, a framework used for building cross-platform apps.
  
  Objective-C: The legacy language used by Apple before being largely replaced by Swift.


**Systems & High-Performance Programming**: These lower-level languages offer fine-grained hardware and memory control, ideal for operating systems and game engines.

  C: The foundational "mother of modern programming languages," used for operating systems and embedded systems.

  C++: An extension of C that adds object-oriented features, dominating game development (Unreal Engine).

  Rust: A modern systems language favored for its strict safety controls and memory performance without a garbage collector.

  Go (Golang): Created by Google for high-performance network services, cloud software, and devops.

  Assembly: A low-level language that directly maps readable mnemonics to machine code instructions.


**Data Science, AI, & Database Management**: Specialized tools designed for processing millions of data rows, statistical computing, and writing artificial intelligence scripts.

  SQL: The global standard query language used to interact with and manage relational databases.
  
  R: A language explicitly focused on statistical computing, heavy data analysis, and advanced charting.
  
  Julia: Engineered for rapid, high-performance numerical and scientific computing.
  
  MATLAB: A proprietary language used heavily by engineers and scientists for matrix math and simulations


**Functional & Academic Languages**: Languages utilizing distinct mathematical programming paradigms, favored in research, finance, and complex engineering systems.

  Haskell: A purely functional programming language utilizing rigorous static typing and lazy evaluation.
  
  Scala: Synthesizes object-oriented and functional concepts on top of the Java Virtual Machine.
  
  Lisp: One of the oldest families of languages, heavily relied upon in early AI development.

  Clojure: A modern, functional dialect of Lisp designed to run seamlessly on the Java ecosystem.
  
  F#: A functional-first platform language designed for complex mathematical modeling within .NET.
  
  Erlang / Elixir: Designed for high-concurrency systems requiring massive uptime, like telecommunication networks.

  Prolog: A logic-based programming language primarily utilized in computational linguistics and artificial intelligence.


**Legacy & Historical Languages**: Though rarely chosen for new projects today, billions of lines of code are written in these languages to run global banking, business, and military systems.

  COBOL: Common Business-Oriented Language, still the backbone of many banking, finance, and legacy enterprise backends.
  
  Fortran: Formula Translation, a legendary computing language optimized for intensive scientific math.

  Pascal: A structured, classic language heavily utilized in the late 20th century to teach sound design principles.

  Ada: A highly secure, structured language designed for the US Department of Defense for mission-critical hardware.

  MUMPS: the heavy-duty, highly active backbone powering the vast majority of modern healthcare IT infrastructure.
